summary sesi-02

pada sesi 02 ini difokuskan belajar mengenai dasar golang yang mencakup : 
1. Conditions
	 = conditions temporary variable, switch, switch with relational operators, switch fallthrough keyword, 
     nested, conditions.
2. Loopings
	 = first way of looping, second way of looping, third way of looping, looping(break and continue keywords),
     nested looping, looping (label).
3. Array
	 = array, array (modify element through index), array (loop through elements), array (multidimensional array).
4. Slice
	 = slice, slice (make function), slice (append function), slice (append function with ellipsis), 
     slice (copy function), slice (slicing), slice (combining slicing and append), slice (backing array), 
     slice (cap function), slice (creating a new backing array).
5. Map 
    = map, map (looping with map), map (deleting value), map (detecting a value), map (combining slice with map).
6. Aliase 
    = aliase 
7. Strings in Depth
    = looping over string (byte-by-byte), looping over string (rune-by-rune).
